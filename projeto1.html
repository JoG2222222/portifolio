Projeto 1: Site da loja fictícia: Hoop & Slam!

Este projeto eu desenvolvi um pequeno site com algumas pequenas funcionalidades de um site de uma loja. Contendo desde uma aba de funcionários até uma aba com o estoque e produtos que podem ser comprados. Neste projeto utilizei conceitos de html e css aprendidos no curso do SENAC. Desde tabelas estilizadas até a distribuição semantica no html do site.
